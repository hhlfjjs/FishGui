template<class ITEM>
class FG_LoopIterator< ITEM >

循环迭代器类.

循环迭代器从起始对象指针开始遍历，直到容器中的最后一个对象， 然后再从容器中的第一个元素开始遍历，直到起始对象（不包括起始对象）

参数:
ITEM 节点中保存的对象类型
