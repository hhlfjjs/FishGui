void FG_Screen::OnDrawRequest (  FG_Message *  pMsg  )  [protected]
 

窗口发送的要求重画消息FG_MSG_DRAW_REQUEST的处理函数

对于窗口对象发来的重画请求，并不直接重画，而是启动一个时间为SCREEN_DRAW_INTERVAL（50毫秒）的定时器， 定时器到期时才执行重画操作，该定时器在删除之前只能被成功一次，因此50 毫秒内的只会发生一次重画操作， 提高了重画的效率，不会引起闪烁感

参数:
pMsg 指向要求重画消息结构的指针

在文件FG_Screen.cpp第179行定义。

参考 SCREEN_DRAW_INTERVAL、FG_Object::StartTimer()及TIMERID_DRAW.
