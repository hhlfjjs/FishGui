bool FG_Window::RegisterKey (  const WORD  wOrgKey,
const WORD  wActKey,
const WORD  wID
) 
 

注册快捷键

快捷键的含义为：当本窗口为焦点窗口时，如果用户按键值为wOrgKey的键盘消息， 直接把该键盘消息发送给标识为wID的控件，并把键值转换成wActKey

参数:
wOrgKey 原始的虚拟键码
wActKey 发送到对应的控件时，应转换成的实际虚拟键码
wID 注册快捷键的控件
返回:
删除操作是否成功
true　 成功
false　失败
