void FG_X11Adaptor::SetXpmPath (  const char *  pPath  ) 
 

设置位图文件所在目录的绝对路径.

两种方法得到FishGUI主目录的绝对路径

  • 从环境变量HOME_FISHGUI中得到
  • 从程序运行时argv[0]所带的运行程序路径和当前路径组合得到

第二种方法在不同的环境中不能保证一定能得到位图文件所在目录的绝对路径， 这种情况下用户必须设置环境变量HOME_FISHGUI

参数:
pPath argv[0]所带的运行程序路径
